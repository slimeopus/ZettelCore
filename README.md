# ZettelCore

ZettelCore — это простой инструмент для управления заметками по методу Zettelkasten с поддержкой тегов, вики-ссылок и визуализации связей между заметками.

## Особенности

- 📝 Создание заметок через CLI
- 🔍 Поиск заметок по тегам
- 🔗 Поддержка вики-ссылок (`[[название_заметки]]`)
- 📊 Визуализация графа связей между заметками (TUI + Rich)
- 🗂 Хранение заметок в формате Markdown с YAML frontmatter
- 🔎 Интерактивный поиск с помощью `fzf` (опционально)

## Установка

1. Убедитесь, что у вас установлен Python 3.8+
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Установите `fzf` для интерактивного поиска (рекомендуется):

```bash
# Linux/macOS (через Homebrew)
brew install fzf

# Или следуйте инструкциям: https://github.com/junegunn/fzf#installation
```

## Использование

### Создание заметки

```bash
python main.py create "Содержимое новой заметки" --tags "тег1,тег2"
```

Пример:
```bash
python main.py create "Изучаю Python и Zettelkasten" --tags "python,learning"
```

### Поиск по тегу

```bash
python main.py find #тег
```

Пример:
```bash
python main.py find #python
```

### Визуализация графа заметок

```bash
python main.py graph
```

Откроется TUI-интерфейс с древовидным представлением связей. Используйте:
- `q` — выход
- Кнопка "Refresh" — обновить граф
- Кнопка "Exit" — закрыть приложение

Если `textual` не установлен, будет использован упрощённый вывод в консоль.

### Интерактивный поиск (с fzf)

Если установлен `fzf`, можно искать и открывать заметки интерактивно.

```bash
python main.py fzf
```

Или по тегу:
```bash
python main.py fzf #python
```

## Формат заметок

Заметки хранятся в папке `notes/` в формате `YYYY-MM-DD-HHMMSS.md` и содержат YAML frontmatter:

```markdown
---
date: '2026-04-08T10:00:00'
tags:
  - python
  - test
---

Содержимое заметки...
```

Вы можете добавлять ссылки на другие заметки с помощью синтаксиса `[[название_заметки]]` (например, `[[2026-04-08-100000]]`).

## Структура проекта

```
ZettelCore/
├── main.py               # Основной CLI-интерфейс
├── config.yaml           # Конфигурация (пока пуст)
├── requirements.txt      # Зависимости
├── cli/                  # Команды CLI
├── core/                 # Ядро приложения
│   ├── storage.py        # Сохранение заметок
│   ├── search.py         # Поиск по тегам
│   ├── graph.py          # Построение графа связей
│   ├── tui.py            # Визуализация графа (Textual + Rich)
│   └── tags.py           # Работа с тегами (заглушка)
├── notes/                # Хранилище заметок
└── README.md             # Этот файл
```

## Зависимости

- `click` — CLI-фреймворк
- `PyYAML` — парсинг YAML frontmatter
- `rich` — красивый вывод в консоль
- `textual` — TUI-приложение для визуализации графа
- `fzf` — интерактивный поиск (устанавливается отдельно)
